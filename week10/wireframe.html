Project Proposal

Personal/ Professional Portfolio

Description

For my project proposal, I would like to make a fully responsive personal/ portfolio website to showcase more of my selected works and projects. I would also like to add a professional section where I can add my resume along with an “about me” section to give more information from a casual point of view.

Purpose

I feel that I still need to refine and expand my skills in HTML, CSS, JavaScript, and the principles of design. This will also help me to prepare for a professional career as a web developer as I want this to be something I can include in my experience when applying to a job.

Audience

My audience will be those whom I will work with in a professional setting both superiors and co-workers.

Data Structures

I will use local JSON files for pages that will be regularly updated.
